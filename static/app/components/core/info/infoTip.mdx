`InfoTip` renders an info icon that reveals a tooltip on hover or focus. It's keyboard accessible out of the box and provides a consistent pattern for supplementary help.

<Storybook.Demo>
  <Flex gap="xs" align="center">
    <Text>Retention Period</Text>
    <InfoTip title="The number of days event data is stored before being automatically deleted." />
  </Flex>
</Storybook.Demo>
```jsx
<Flex gap="xs" align="center">
  <Text>Retention Period</Text>
  <InfoTip title="The number of days event data is stored..." />
</Flex>
```

## Size

The `size` prop controls the icon size. Available sizes are `xs`, `sm` (default), and `md`.

```jsx
<InfoTip title="Extra small" size="xs" />

```

## When to Use

Use `InfoTip` to:

- Add context to headings or section titles
- Show supplementary information without inline text
- Explain settings or configuration options

Prefer `InfoTip` over adding a `Tooltip` to an arbitrary icon.

## Accessibility

- **Keyboard accessible**: Users can Tab to focus the icon and reveal the tooltip
- **Focus indicator**: A visible focus ring appears when focused
- **Screen reader support**: The icon has `role="img"` and `aria-label="More information"`
